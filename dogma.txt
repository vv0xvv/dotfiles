### The Notes - QuickRef


C-x C-s   Save
C-x s     Save some buffers (...don't use)
C-x C-c   Close session

C-x C-b   List buffers
C-x b     Switch buffer
C-x 1     Close all but one window

C-x u  OR  C-/      undo
C-s       Search forward
C-r       Search reverse

M-w       Copy text
C-w       Cut text
C-y       Yank text (paste, chronological)
C-k       Kill-line
INS       Insert/overwrite



M-x revert-buffer      Revert changes since saved
C-x i      Insert file content
